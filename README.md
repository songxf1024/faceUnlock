# faceUnlock
Low-cost implementation of face recognition to unlock Windows.

# Environments
## Hardware
- Digispark Attiny85
- USB Serial

## Software
- [Arduino](https://www.arduino.cc/en/Main/Software)
- [Python](https://www.python.org/downloads/)
- Opencv
- dlib
- pyserial
> - 安装库：`pip install dlib-bin opencv-python pyserial`
