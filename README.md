# Environments
## Hardware
- Digispark Attiny85
- USB Serial


## Software
- [Arduino](https://www.arduino.cc/en/Main/Software)
- [Python](https://www.python.org/downloads/)
- Opencv
- dlib
- pyserial

```bash
pip install dlib-bin opencv-python pyserial
```
